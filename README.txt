
NodeJS, Express and SQLite


	This is an example project based on NodeJS, Express and SQLite database.
In this project, CRUD functions are coded using SQLite database.


Following versions are used in this examle project.

NodeJS - v9.11.1
SQLite 3
Express 4.16.4


Please read [package-lock.json] for rest of versions.

Necessary libs
--------------
npm install express
npm install sqlite3
npm install express
npm install body-parser
npm install path
npm install mustache
npm install opn
npm install consolidate


run as
> node server.js
